## 📝 Prompt Original (Sin iteraciones)

> Eres un desarrollador frontend. Tengo que crear un pequeño ejercicio de HTML y JavaScript, cuya lógica consiste en invertir el orden de una cadena de texto. 
> 
> **UX:** A medida que escribimos en un input de texto, se va mostrando lo escrito de forma inversa. No es necesario un botón de submit; la cadena invertida se genera on the fly. 
> 
> **UI:** Crea un estilo rompedor, como no ha de haber nada más en la pantalla, puedes emplear el 100% del viewport con texto gigante, crear transiciones de color del texto o del background mientras escribes e incluso tonos de audio según la tecla pulsada, todo ello para crear una experiencia interactiva con mucha creatividad. Rienda suelta a la imaginación. 
> 
> **Archivos de salida:** Únicamente dos archivos, `index.html` para la visualización en navegador y `script.js` para la lógica. Puede llamarse a librerías de terceros desde el propio HTML.
